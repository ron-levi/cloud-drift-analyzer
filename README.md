# Cloud Drift Analyzer

A tool for analyzing infrastructure drift between Infrastructure as Code (IaC) definitions and actual cloud resources.

## Features

- Supports multiple cloud providers (AWS, GCP)
- Supports multiple IaC tools (Terraform, Pulumi)
- Real-time drift detection
- Configurable notifications (Slack)
- HTML and JSON reporting
- REST API for integration

## Installation

```bash
poetry install
```
